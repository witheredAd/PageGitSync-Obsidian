# Obsidian Sample Plugin

This is a Publish Helper Plugin on [Obsidian](https://obsidian.md), for blog system [Rinz-Page-Next](https://github.com/witheredAd/rinz-page-next).

Can works with Obsidian Mobile! The git function is provided by [isomorphic-git](https://github.com/isomorphic-git/isomorphic-git).

This project uses TypeScript to provide type checking and documentation.
The repo depends on the latest plugin API (obsidian.d.ts) in TypeScript Definition format, which contains TSDoc comments describing what it does.
